1. console.groq.com → הירשם
2. תפריט שמאלי → API Keys → Create API Key
3. תן שם (לדוגמא "Ogen") → העתק → הדבק כאן
4. שמור → לחץ "בדוק חיבור" → אמור להופיע ✅

💚 Groq חינמי עם מגבלת קצב נדיבה (כמה אלפי בקשות ביום) - אין צורך באשראי.
