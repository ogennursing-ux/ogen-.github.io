# חתימה דיגיטלית (Ogen Sign)

אפליקציית ווב לחתימה דיגיטלית על קבצי PDF — **רצה כולה בדפדפן**, ללא שרת.
הקובץ לעולם לא עוזב את המכשיר של המשתמש.

## יכולות

- העלאת PDF בגרירה או בחירה מהמחשב
- צפייה בכל עמודי המסמך
- הוספת שדות על המסמך ומיקומם בגרירה/שינוי גודל (מגע ועכבר):
  - ✒️ **חתימה** — לוח ציור עם חיתוך רקע אוטומטי
  - 🔤 **טקסט** — תמיכה מלאה בעברית (RTL)
  - 📅 **תאריך**
  - ☑️ **תיבת סימון**
- הורדת PDF חתום חדש בלחיצה אחת

## טכנולוגיה

- **React + Vite**
- **pdf.js** לרינדור עמודי ה-PDF (worker מקומי, ללא CDN)
- **pdf-lib** ליצירת ה-PDF החתום

### למה טקסט עברי מרונדר כתמונה?

הפונטים המובנים של `pdf-lib` תומכים רק ב-WinAnsi ולא בעברית. לכן טקסט ותאריך
מצוירים על `canvas` ומוטבעים כ-PNG שקוף — כך כל תו Unicode עובד, כולל עברית
בכיוון RTL.

## הרצה מקומית

```bash
npm install
npm run dev      # פיתוח
npm run build    # בנייה לפרודקשן (תיקיית dist/)
npm run preview  # תצוגה מקדימה של ה-build
```

## פריסה

`vite.config.js` מוגדר עם `base: './'` כך שה-build עובד גם משורש דומיין וגם
מתת-נתיב (למשל GitHub Pages).
