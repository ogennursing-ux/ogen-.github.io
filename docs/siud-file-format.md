# פורמט קובץ מנות סיעוד (ביטוח לאומי)

מסמך זה מתעד את המבנה של קובץ ה־`oz_siud_manot_<ת"ז>_<מכסה>.txt` — כפי
שפוענח מתוך קובץ דוגמה יחיד. המחולל שבמאגר
(`src/lib/bituachSiudFile.js`) משחזר את הדוגמה **בית‑אחר‑בית**.

## מאפיינים כלליים

| מאפיין | ערך |
|--------|-----|
| קידוד | **ISO‑8859‑8** (עברית, בית בודד לכל תו) |
| סיום שורה | `CRLF` (`\r\n`) |
| מפרידים | תו `Tab` (`0x09`) בין ארבעת המקטעים הראשונים |
| ריפוד | רווחים (`0x20`) מימין, יישור לשמאל, רוחב קבוע |
| מבנה | שורת רשומה אחת + שורת סיכום (trailer) |
| אורך שורת רשומה | 704 בתים |
| אורך שורת trailer | 20 בתים |

השם: `oz_siud_manot_216095568_90.txt` = `oz_siud_manot_<ת"ז>_<קוד מכסה>`.

## שדות שניתן לחלץ ממסמך (דרכון/ת"ז)

היסט = מיקום הבית בשורת הרשומה (מתחיל מ־0). אלה השדות שהמחולל משתיל.

| שדה | היסט | רוחב | סוג | ערך בדוגמה |
|------|------|------|-----|-----------|
| `id` — ת"ז / מזהה | 3 | 9 | ספרות | `216095568` |
| `lastNameLatin` — שם משפחה לועזי | 37 | 19 | ASCII | `KOBILOV` |
| `firstNameLatin` — שם פרטי לועזי | 57 | 51 | ASCII | `FORIGJON` |
| `lastNameHebrew` — שם משפחה עברי | 109 | 18 | עברית | `קובילוב` |
| `firstNameHebrew` — שם פרטי עברי | 128 | 8 ⚠️ | עברית | `` פוריג`ון `` |
| `birthDate` — תאריך לידה | 137 | 8 | `YYYYMMDD` | `19891012` |
| `city` — עיר | 260 | 18 | עברית | `תל אביב - יפו` |
| `street` — רחוב/בניין | 278 | 19 | עברית | `דירה מנחם ארבה` |
| `house` — מספר בית | 297 | 7 | ASCII | `10` |
| `phone` — טלפון | 304 | 10 | ספרות | `0547824652` |

הת"ז מופיעה גם בשורת ה־trailer (באותו היסט 3) והמחולל מסנכרן את שתיהן.

> ⚠️ **רוחב שם פרטי עברי (8):** בדוגמה השם העברי צמוד ישירות לבלוק המספרי
> שאחריו, כך שלא ניתן היה להסיק את רוחבו האמיתי — נבחר 8 (אורך השם בדוגמה).
> אם יתקבל מפרט רשמי עם רוחב אחר, יש לעדכן את `FIELD_MAP.firstNameHebrew.width`
> ב־`src/lib/bituachSiudFile.js`; כל שאר השדות תלויי‑תבנית ושומרים על מקומם.

## שדות אטומים (נשמרים כברירת מחדל מהדוגמה)

לא ניתן לגזור אותם ממסמך זהות, והם דורשים את המפרט הרשמי של ביטוח לאומי.
המחולל שומר עליהם ורבטים מהתבנית:

| מיקום | ערך בדוגמה | פירוש משוער |
|-------|-----------|-------------|
| ראש מקטע 0 | `500` | קוד סוג רשומה |
| אחרי הת"ז | `0` `90` `202315` `FA` `0810944` | קוד מכסה + קודים פנימיים |
| בלוק אחרי ת. לידה | `20055` | לא מפוענח |
| מקטע 4 | `974780700 202604301 026923862` | מספרי תיק/הפניה |
| מקטע 4 | `אוליאל`, `יהודה119415000` | ככל הנראה שם נציג/יישוב + סכום |
| מקטע 4 | `000` | לא מפוענח |
| trailer | `500…09990001` | קוד סיום + מונה רשומות (`0001`) |

```js
  firstNameLatin: 'FORIGJON',
  lastNameHebrew: 'קובילוב',

// bytes → Uint8Array בקידוד ISO-8859-8, מוכן לשמירה כ־oz_siud_manot_216095568_90.txt
```
